Planium

By Ntdi World

Planium is an API to add text onto an image. This API is meant for you to use as it comes with lots of features. Images are cached, so don't worry about spamming the API.

  • Object Orientated
  • Least-Frequently-Used Cache Invalidation
  • AWT image buffering

The source code can be found here.
